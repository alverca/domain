## Usage

### Install

```shell
npm install --save @tokyo-tower/domain
```

```Javascript
var ttts = require("@tokyo-tower/domain");
```

### Environment variables

| Name    | Required | Value         | Purpose |
| ------- | -------- | ------------- | ------- |
| `DEBUG` | false    | ttts-domain:* | Debug   |
